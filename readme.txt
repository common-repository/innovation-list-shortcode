Add [list-posts-innovation] shortcode which enables you to output anywhere in your page or post. 

== Description ==
This plugin provides add Innovations and shortcode to get list Innovation posts with slider.

Here You can get code with using [list-posts-innovation].

== Installation ==

1. Download the plugin.
2. Upload the whole plugin directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Now, go to the page/post menu add Shortcode [list-posts-innovation]
4. Enjoy!
